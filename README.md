# 🍔 麥當勞台灣最划算比價神器

> 一站式比較台灣麥當勞所有餐點價格、套餐方案與優惠活動，自動計算最划算的點餐組合。

**🌐 Demo：** 本機啟動 `python3 -m http.server 8080` 後開啟 http://localhost:8080

---

## 📌 功能總覽

### 1. 📋 完整菜單
- 顯示所有主餐（牛肉堡、雞肉堡、海鮮、早餐等）的**單點價格**與**套餐價格**
- 自動計算每道餐點套餐可省多少錢
- 支援**關鍵字搜尋**（例：大麥克、雞腿堡）
- 支援分類篩選（經典牛肉、雞肉堡、安格斯、雞塊…）
- 涵蓋薯條/配餐、飲料（碳酸/茶飲/咖啡/果汁）、甜點

### 2. 🏷️ 優惠方案總覽
列出 10 種主要優惠方案並說明省錢幅度：

| 優惠 | 說明 | 省錢幅度 |
|---|---|---|
| **甜心卡 (App限定)** | A區主餐搭B區商品半價 | B區最高省50% |
| **1+1=$50** | 指定商品任選2件$50 | 省$0~$18 |
| **超值全餐** | 主餐+中薯+中飲 | 省$7~$18 |
| **App 隨享券** | 不定期買一送一/折扣 | 最高省50% |
| **早安優惠** | 早餐時段套餐 | 最低$79起 |
| **分享盒** | 多人分享人均更省 | 人均省$20+ |
| **行動支付回饋** | LINE Pay/街口等 | 最高10%回饋 |
| **快樂兒童餐** | 含玩具的兒童套餐 | 含玩具超值 |
| **歡樂送外送優惠** | 滿額折/免外送費 | 依活動而定 |
| **銅板輕鬆點** | $18起的超值單品 | 最低$18 |

### 3. 📊 套餐比較（視覺化長條圖）
- 點選任何一個主餐
- 系統自動比較「全部單點 vs 套餐 vs 甜心卡」三種方式
- 視覺化長條圖顯示價差
- 附詳細情境表（只買主餐、加小/中飲、套餐、升級大薯大飲）
- 綠色標記最划算選項

### 4. 🧮 最佳點餐方案計算器（核心功能）
1. 選擇想吃的**主餐**（可+/−數量，支援多個）
2. 選擇**配餐**（薯條等）
3. 選擇**飲料**
4. 選擇**甜點/其他**
5. 按下「計算最划算方案」按鈕

系統同時計算 **5種策略**：

| 策略 | 說明 |
|---|---|
| 方案A：全部單點 | 基準對照 |
| 方案B：儘量用套餐 | 主餐配中薯+中飲走套餐價 |
| 方案C：甜心卡策略 | 主餐(A區)+最貴B區商品半價 |
| 方案D：1+1=$50策略 | 小品項走1+1=$50，其餘套餐 |
| 方案E：綜合最佳策略 | 套餐+甜心卡+1+1混搭 |

自動排序，標示最划算方案與節省金額。

**範例：大麥克 + 冰紅茶(中)**
- 全部單點：$75 + $35 = **$110**
- 超值套餐：**$135**（含中薯，若不想吃薯則不划算）
- 甜心卡：$75 + $18（紅茶半價）= **$93** ← 最划算

## 📦 技術規格

- **純前端**：單一 HTML 檔案，無需後端、無需安裝任何套件
- **無外部依賴**：所有樣式和邏輯皆內嵌
- **RWD 響應式**：支援手機、平板、桌機
- **資料格式**：所有價格以 JSON 物件儲存於 `menuData` 常數，方便更新

## 📝 資料更新說明

所有價格定義於 `index.html` 的 `menuData` 物件：

```js
const menuData = {
  mains: [
    { id: 'm01', name: '大麥克', category: '經典牛肉', single: 75, combo: 135 },
    // ...
  ],
  sides: [...],
  drinks: [...],
  desserts: [...]
};
```

> ⚠️ **免責聲明**：價格資料根據公開資訊整理，僅供參考。實際價格以台灣麥當勞門市公告為準，建議點餐前至[麥當勞官網](https://www.mcdonalds.com/tw/zh-tw.html)或 App 確認。
